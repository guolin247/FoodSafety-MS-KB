The **Food Safety MS Knowledge Base** is a structured repository of mass spectrometry (MS) parameters extracted exclusively from official regulatory standards (China GB, US USDA/FDA, EU CEN/EURL). 

Unlike generic spectral libraries, this dataset bridges the gap between **regulatory compliance documents** and **analytical data**, providing method-linked metadata (sample preparation, chromatography conditions, and validation parameters).
